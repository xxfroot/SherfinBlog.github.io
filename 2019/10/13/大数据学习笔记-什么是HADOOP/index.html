大数据学习笔记:什么是HADOOP

学习大数据有段时间了，一直没有时间来好好总结自己所学的东西，所以现在从零开始，总结复习我所知道的关于大数据的一切！
我想大部人都和我一样，涉足大数据行业的第一步就是学习Hadoop,那么我现在想自我反问到底什么是Hadoop？
依据官网上所讲:

Hadoop其实是一个数据处理工具，具有可以以高可用布式集群的方式，存储，计算，分析大规模数据的能力。

上图同样来自Hadoop的官网，上面粗略的说明了了Hadoop框架的五大模块
Hadoop Commons:Hadoop组件的公共单元，用以支撑Hadoop其他模块的运行。
Hadoop Distributed File System (HDFS™):Hadoop分布式文件存储系统。
Hadoop YARN：主要用于Hadoop集群，计算时的作业调度以及集群的资源管理。
Hadoop MapReduce：基于Yarn的大规模数据并行处理计算框架。
Hadoop Ozone：Hadoop对象存储（研发中）。
Hadoop Submarine：为Hadoop提供机器学习的能力（研发中）。
